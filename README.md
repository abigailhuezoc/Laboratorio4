# Laboratorio_4 - API REST con Flask
Este proyecto implementa una API REST para adminsitrar el
catálago de productos de TecnoMarket en memoria.

## Instrucciones para ejecutar la aplicación

1. Active el entorno virtual (`.venv`).
2. Instale los requerimientos del sistema:
   ```bash
   pip install -r requirements.txt
   ```
3. Inicie el servidor de la API:
   ```bash
   python app.py
   ```
4. En otra terminal, ejecute el programa cliente para verificar el consumo:
   ```bash
   python cliente.py
   ```
